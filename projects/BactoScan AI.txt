🧫 BactoScan AI – Automated Bacterial Colony Counter
Overview

BactoScan AI is an AI-powered mobile application developed to automate bacterial colony detection and counting from Petri dish images. The platform helps microbiology researchers, laboratory technicians, educators, and students perform colony analysis within seconds, significantly reducing manual effort and human counting errors.

The application combines Artificial Intelligence, Cloud Computing, Mobile Development, and Secure Payment Processing into a single ecosystem, providing a fast, scalable, and user-friendly laboratory solution.

Problem Statement

Traditional bacterial colony counting is a time-consuming and error-prone process. Researchers often spend significant time manually counting colonies, especially when working with large datasets or multiple Petri dishes.

BactoScan AI addresses this challenge by using a custom-trained YOLOv8 object detection model to automatically identify and count bacterial colonies with high accuracy.

System Architecture
1. User Authentication

Users register using their phone number. Firebase Authentication securely verifies users through OTP-based authentication.

2. Image Processing Workflow
User captures a Petri dish image.
Image is sent to the cloud inference server.
YOLOv8 model performs colony detection.
Detection coordinates and colony counts are returned.
Results are displayed instantly within the application.
3. Data Synchronization

User profiles, scan history, and credit management are synchronized using Firebase Firestore and monitored through a Django backend service.

4. Premium Access Management

Users can purchase additional scan credits or premium subscriptions through Razorpay's secure payment infrastructure.
